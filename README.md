🔹 Telecom X – Previsão de Churn (Parte 2)
Projeto de Machine Learning com foco na previsão de clientes com maior risco de evasão, evoluindo a análise exploratória para uma abordagem preditiva.

🎯 Objetivo:
Antecipar churn e apoiar estratégias de retenção baseadas em dados.

🤖 Modelagem:

Regressão Logística (baseline interpretável)
Random Forest (melhor desempenho)

📊 Etapas do projeto:

Preparação e tratamento dos dados
Engenharia de atributos
Treinamento e avaliação de modelos
Análise de importância das variáveis

🔎 Principais insights:

Clientes com contratos mensais apresentam maior risco de churn
Baixo tempo de permanência é um forte indicador de evasão
Métodos de pagamento influenciam a retenção

💡 Impacto no negócio:

Possibilita ações preventivas de retenção
Apoia decisões estratégicas com base em previsão de risco
Reduz potencial perda de receita recorrente

🛠️ Tecnologias: Python, Pandas, Scikit-learn

🔗 https://github.com/Johnsantos1989/Challenge_Telecom_X_Parte2
